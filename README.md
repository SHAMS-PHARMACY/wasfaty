# wasfaty (Shams Pharmacies)

Central landing for "Wasfaty" — auto-detect user's location and route to the nearest branch (≤ 1 km) via WhatsApp.

## Files
- `index.html` — Landing page (Arabic, RTL)
- `style.css` — Minimal styling
- `script.js` — Geolocation + nearest branch selection + redirect logic
- `branches.json` — Branches data (name/address/lat/lon/whatsapp/maps_url)

## Deploy (GitHub Pages)
Settings → Pages → Source: `main` / `(root)` → Save
Then open: `https://shams-pharmacy.github.io/wasfaty/`
